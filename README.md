## Block Odyssey Frontend Task

### 시작하기

1. 의존성 설치
```shell
npm install // yarn
```

2. 로컬 서버 실행
```shell
npm run start // yarn start
```

### 주 사용 기술

> 💡 본 프로젝트는 Node v14 환경 기준으로 개발되었습니다.

- Typescript v4.9.4
- React v18.2.0
- Redux v8.0.5
- Redux-toolkit v1.9.1
- Node-sass v8.0.0
